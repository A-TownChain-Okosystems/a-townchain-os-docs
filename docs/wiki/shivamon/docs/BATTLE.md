# Battle-System

## Rundenbasierter Kampf
1. Initiativ-Berechnung: (SPD + random(0,20))
2. Angriff: ATK - Gegner DEF (min. 1 Schaden)
3. SP-Angriff: SP_ATK vs SP_DEF
4. Typ-Effizienz: Feuer > Pflanze > Wasser > Feuer
5. Sieg: Gegner HP = 0

## Rewards
- Sieg: +50 XP, +1 WIN, Loot (zufällig)
- Niederlage: +10 XP, +1 LOSS

## Level-Up
- Level 1-10: 100 XP/Level
- Level 11-50: 200 XP/Level
- Level 51-100: 500 XP/Level
