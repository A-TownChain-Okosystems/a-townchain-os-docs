# UI-Komponenten

## WalletPanel
- Balance anzeigen (ATC + FFT)
- Send-Dialog mit Signatur
- QR-Code Generator

## ChainExplorer
- Block-Liste (live)
- TX-Suche nach Hash/Adresse
- Node-Map

## ShivamonGallery
- Grid-View aller eigenen Shivamon
- Rarity-Filter
- Battle starten

## GeminiChat
- BYOK (Bring Your Own Key)
- ATCLang generieren
- Contract erklären

## GatewayStatus
- Health-Check Port 4000
- Rate-Limit-Anzeige
- Latenz-Graph
