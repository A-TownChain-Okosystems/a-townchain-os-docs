# ATC Franchise Factory

**Version:** v1.0.0 | **Status:** Beta

## Ubersicht
Das Franchise-Modul ermoglicht die dezentrale Erstellung und Verwaltung
von ATC-Franchise-Lizenzen auf der Blockchain.

## Features
- Franchise-Lizenz als NFT (ATC-9000)
- Automatische Royalty-Verteilung
- DAO-gesteuerte Franchise-Regeln
- On-Chain Governance
